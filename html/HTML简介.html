HTML简介

什么是HTML?

  • HTML（Hyper Text Markup Language）是超文本标记语言的缩写。
  • HTML使用标记语言描述Web页面的结构。
  • HTML元素是HTML页面的构建块。
  • HTML元素通过标签tag表示。
  • HTML标签是“标题”、“段落”、“表格”等内容的一部分。
  • 浏览器不显示HTML标签，但是使用它们来渲染页面内容。

一个简单的HTML文档

  <!DOCTYPE html>
  <html>
  <head>
    <title>无聊的网站</title>
    <meta charset="utf-8">
  </head>
  <body>

    <h1>这个是标题</h1>
    <p>这个是段落.</p>

  </body>
  </html>

实例解析

  • <!DOCTYPE html> 声明定义这个文档是HTML5
  • <html> 元素是HTML页面的根元素
  • <head> 元素包含关于文档的元数据信息
  • <title> 元素为文档指定一个标题
  • <body> 元素包含可视化页面内容
  • <h1> 元素定义一个大的标题
  • <p> 元素定义一个段落

HTML 标签

HTML标签是被尖括号包围的元素名称

<标签名称> 内容... </标签名称>
  • HTML标签通常成对出现，例如<p>和</p>
  • 在这个标签对里，第一个是开始标签，第二个是结束标签
  • 结束标签和开始标签相同，但是在标签名称前面需要插入一个反斜线

提示：开始标签页称为开标签，结束标签页称为闭标签
