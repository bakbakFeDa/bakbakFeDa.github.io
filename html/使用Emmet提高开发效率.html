使用Emmet提高开发效率

Emmet简介

Emmet 是一个Web工具，可以极大的提高html和css的编写效率。

官网：https://docs.emmet.io/

常用语法

  1. 叹号!生成html基本模板。

  2. 子元素>

    div>ul>li
    <div>
    <ul>
        <li></li>
    </ul>
</div>

  3. 兄弟元素+

    div+p+dp
    <div></div>
<p></p>
<blockquote></blockquote>

  4. 嵌套子元素^

    div+div>p>span+em
    <div></div>
<div>
    <p><span></span><em></em></p>
</div>

  5. 多次重复*

    ul>li*5
    <ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>

  6. 分组()

    div>(header>ul>li*2>)+footer>p
    <div>
    <header>
        <ul>
            <li><a href=""></a></li>
            <li><a href=""></a></li>
        </ul>
    </header>
    <footer>
        <p></p>
    </footer>
</div>

  7. 属性操作 id 和 class

    div#header+div.page+div#footer.class1.class2.class3
    <div id="header"></div>
<div class="page"></div>
<div id="footer" class="class1 class2 class3"></div>

    8. 语法：elem#id elem.class
