for 循环

1. 语法

Solidity 中， for循环的语法如下：


for (初始化; 测试条件; 迭代语句) {
   // 如果表达式的结果为真，就循环执行以下语句
   ......
}

 

2. 示例

下面的示例代码定义了一个合约 Example，在这个合约中定义了一个私有变量 counter ，一个mapping类型的变量 numbers，以及两个函数 setCounter 和 fillNumbers。

setCounter 函数用来设置 counter 的值，这是一个公有函数，可以在外部调用。

fillNumbers 函数用来将0~counter-1之间的数字乘以2并存入mapping中，最后返回最后一个数字。这个函数使用了一个 for 循环来实现。在循环中，定义了一个变量 i ，从0开始，每次增加1，直到 i 等于 counter。在循环体中，我们使用了 numbers[i] = i*2; 来将数字 i 乘以 2 并存入 mapping 中。由于我们从0开始循环到counter-1，所以最后mapping中将会存储0到counter-1之间所有数字乘以2的结果。

最后, fillNumbers函数返回最后一个数字，即 numbers[counter-1]。

这样编写的程序可以在合约中完成指定次数的操作，并在运行中记录和存储结果。

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract Example {
    uint private counter;
    mapping(uint => uint) public numbers;

    function setCounter(uint _counter) public {
        counter = _counter;
    }

    function fillNumbers() public payable returns(uint) {
        for(uint i = 0; i < counter; i++) {
            numbers[i] = i * 2;
        }
        return numbers[counter-1];
    }
}

需要注意的是在 solidity 中 for 循环是有限制的，每次循环都需要消耗一定的 gas，所以在编写合约时需要谨慎使用循环，特别是在循环次数很多的情况下。如果循环次数过多，可能会导致合约执行超时或者消耗过多的 gas，从而导致交易失败。
