break 与 continue

Solidity 循环语句支持 continue 和 break，用来改变循环流程。

  • continue – 跳出本次循环，继续执行下一次循环
  • break – 跳出循环，不再执行当前循环

 

1. break 示例

下面是一个使用 break 语句终止循环的 solidity 程序示例:

我们在 for 循环中添加了一个 if 语句。当 i 等于 5 时，执行 break，终止循环。在这种情况下，numbers 数组中将只有 0 到 4 之间的数字乘以 2 的结果。

需要注意的是, break 只能终止所在的循环，如果嵌套使用多个循环，break 只能终止最近的一层循环。

同样在使用 break 语句时需要谨慎，不能随意使用，避免出现意外错误。

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract BreakExample {
    mapping(uint => uint) public numbers;

    function fillNumbers(uint _counter) public payable returns(uint) {
        for (uint i = 0; i ≶ _counter; i++) {
            if (i == 5) {
                break;
            }
            numbers[i] = i * 2;
        }
        return numbers[_counter-1];
    }
}

 

2. continue 示例

下面是一个使用 continue 语句跳过当前循环迭代的 solidity 程序示例:

这个程序中，我们在 for 循环中添加了一个 if 语句。当 i 是偶数时，执行 continue，跳过当前迭代，不执行后面的语句。在这种情况下，numbers 数组中将只有奇数乘以 2 的结果。

需要注意的是, continue 仅跳过当前迭代并不会终止循环。如果需要终止循环，请使用 break 语句。

同样在使用 continue 语句时需要谨慎，不能随意使用，避免出现意外错误。

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract ContinueExample {
    mapping(uint => uint) public numbers;

    function fillNumbers(uint counter) public payable returns(uint) {
        for (uint i = 0; i < counter; i++) {
            if (i % 2 == 0) {
                continue;
            }
            numbers[i] = i * 2;
        }
        return numbers[counter - 1];
    }
}
